# Team-Weathy

### 커밋 컨벤션
| 카테고리 | 설명 |
| --- | --- |
| 🪄 Chore | 개발 환경 세팅, 빌드 수정, 패키지 추가, 환경변수 설정 |
| 🌎 Deploy | 배포 |
| 📝 Docs | 문서 수정, 리드미 업데이트 |
| 🚀 Feat | 새로운 기능 추가 또는 기능 업데이트 |
| 🔥 Hotfix | 치명적인 버그 수정 |
| 🎨 Design | CSS 등 UI 디자인 변경 |
| 🔨 Fix | 버그 또는 에러 수정 |
| ⭐️ Style | 코드 포맷팅, 코드 오타, 함수명 수정 등 스타일 수정 |
| 🧠 Refactor | 코드 리팩토링 |
| 📁 Rename | 파일 혹은 폴더명 수정 혹은 이동 |
| ⛔️ Remove | 파일 삭제 |
